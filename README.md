##  Part A — Concept Application

Problem Identification:
- Classified real-world problems into:
  - Supervised Learning (Regression / Classification)
  - Unsupervised Learning
  - Reinforcement Learning

Examples:
- Spam detection → Supervised (Classification)
- Customer grouping → Unsupervised
- Robot learning → Reinforcement
- House price prediction → Supervised (Regression)
- Product recommendation → Unsupervised

Regression vs Classification:
- Regression → predicts continuous values (e.g., house price)
- Classification → predicts categories (e.g., spam detection)

Dataset Understanding:
- Created a Pandas DataFrame
- Separated:
  - Features (X)
  - Target (y)

##  Part B — Examples

Regression:
- Input: numerical features (area, rooms)
- Output: continuous value (price)
- Use case: house price prediction

Classification:
- Input: text or numerical features

- Use case: email filtering
